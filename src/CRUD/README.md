# 对象数组的 CRUD（增删查改）

定义一个长度为 3 的对象数组，数组存储 1～3 名学生对象作为初始数据。  
学生对象的学号各不相同。

## 学生的属性

学号、姓名、年龄

## 要求

1. 再次添加一个学生对象，并在添加时进行学号的唯一性判断；  
  如果学号已经存在，提示不能重复添加
2. 添加完毕后，遍历所有学生信息
3. 通过学号删除学生信息
  如果存在，则删除；如果不存在，则提示删除失败
4. 删除完毕后，遍历所有学生信息
5. 查询数组学号为 "002" 的学生信息
  如果存在，则将他的年龄 +1 岁
